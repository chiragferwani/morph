# Changelog

All notable changes to the morph project will be documented in this file.

## [0.1.0] - 2026-07-11

### Added
- Initial release of the morph web scraping package
- **Scrapers**: text, images, audio, and video extraction
- **Converters**: image-to-text (OCR), audio-to-text, video-to-audio/text/images
- **CLI**: Full command-line interface with scrape, convert, and web commands
- **Web Interface**: Dark-themed responsive HTML/CSS/JS interface with Flask backend
- **Export**: Support for TXT, CSV, JSON, and ZIP output formats
- **Utils**: URL validation, file downloading, and data export helpers
- **Tests**: Unit tests for scrapers, converters, and utilities
- **Documentation**: Installation, usage, API, CLI, developer, and publishing guides
- **Packaging**: setup.py, pyproject.toml, MANIFEST.in for PyPI publishing
